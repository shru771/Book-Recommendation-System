# Book-Recommendation-System
Content-based Book Recommendation System using Python, Streamlit, TF-IDF, and Cosine Similarity.

This project recommends books using TF-IDF Vectorization and Cosine Similarity.

## Technologies Used
- Python
- Streamlit
- Pandas
- Scikit-learn

## Features
- Book recommendation
- Search functionality
- Rating analysis
- Data visualization
